# Rick and Morty app

Running the app:

```
npm install
npm start
cordova run ios
```

To build your app for deployment, run:

```
npm run build
cordova build ios
cordova run ios 
// or
cordova run android 
```

The build folder will then contain all of your app's files, optimized and ready for deployment.
